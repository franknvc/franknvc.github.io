How I created my blog using Hugo

In the last few years, static site generators have gathered lots of attention with the like of Hugo, Gatsby, etc… These static site generators are mostly used for informative websites such as blog with features that help user to focus more on writing contents.

The most popular generator is Hugo, which is written in the Go language and claimed to have the fastest speed among all generators. As the result, Hugo was chosen to be the generator for this blog.
