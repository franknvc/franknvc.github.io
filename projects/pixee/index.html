Pixee

Apr. 1, 2021

A simple Pixel Art Maker written in Python. With this tool, you will be able to create pixel art quickly and save them for further use in the future. The GUI is designed using the tkinter library.

Tech used

figure 1

The window consists of a drawing canvas in grid style which simulates the pixels and a button on the top left corner which opens a color palette.

figure 2

Color palette allows you to choose new color for drawing. In the panel, there is an X marker, a color grid, and a previewer of the chosen color as well as its color code.

figure 3

A simple demonstration. The user just need to click on each square to paint.
